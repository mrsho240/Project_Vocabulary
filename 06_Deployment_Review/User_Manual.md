# คู่มือการใช้งาน (User Manual)
**Vocabulary Quiz Generator: ระบบสร้างแบบทดสอบคำศัพท์อัตโนมัติ**

---

## 1. บทนำ (Introduction)
**Vocabulary Quiz Generator** เป็นแอปพลิเคชันสำหรับฝึกฝนและทดสอบความรู้ด้านคำศัพท์ภาษาอังกฤษผ่านแบบทดสอบปรนัย (Multiple Choice) ผู้ใช้งานสามารถเลือกระดับความยากและรับทราบผลคะแนนได้ทันที เพื่อช่วยพัฒนาทักษะทางภาษาได้อย่างมีประสิทธิภาพ

---

## 2. การเริ่มต้นใช้งาน (Getting Started)

### 2.1 ความต้องการของระบบ (System Requirements)
* **Operating System:** Windows, macOS หรือ Linux
* **Software:** ต้องติดตั้ง **Java Runtime Environment (JRE)** เวอร์ชั่น 8 ขึ้นไป

### 2.2 การเปิดโปรแกรม
1.  ตรวจสอบว่าเครื่องคอมพิวเตอร์ติดตั้ง Java เรียบร้อยแล้ว
2.  ดับเบิ้ลคลิกที่ไฟล์โปรแกรม (เช่น `Vocabulary Quiz Generator.jar`) เพื่อเริ่มต้นใช้งาน

---

## 3. ส่วนประกอบและการใช้งาน (Features & Usage)

### 3.1 หน้าเมนูหลัก (Main Menu)
เมื่อเข้าสู่โปรแกรม จะพบกับหน้าจอหลักที่เรียบง่ายและเป็นมิตร ประกอบด้วยเมนูสำคัญดังนี้:

* **Start:** ปุ่มสีฟ้าขนาดใหญ่ตรงกลาง ใช้สำหรับ **"เริ่มทำแบบทดสอบ"**
* **Vocabulary:** สำหรับเข้าไปดูรายการคำศัพท์ทั้งหมด (คลังคำศัพท์)
* **About Me:** สำหรับดูข้อมูลผู้พัฒนาโปรแกรม

> **วิธีใช้งาน:** คลิกที่ปุ่ม **"Start"** เพื่อเริ่มเข้าสู่กระบวนการทดสอบ

---

### 3.2 การเลือกระดับความยาก (Select Difficulty)
ก่อนเริ่มทำข้อสอบ ผู้ใช้ต้องเลือกระดับความยากที่ต้องการทดสอบ โดยมี 3 ระดับให้เลือก:

1.  **Beginner:** คำศัพท์พื้นฐาน ง่ายต่อการเริ่มต้น
2.  **Moderate:** คำศัพท์ที่ใช้ในชีวิตประจำวัน
3.  **Advanced:** คำศัพท์เชิงวิชาการและคำศัพท์ชั้นสูง

* **Back to Main Menu:** หากต้องการยกเลิก สามารถกดปุ่มนี้ที่มุมซ้ายล่างเพื่อกลับไปหน้าแรก

---

### 3.3 การทำแบบทดสอบ (Quiz Interface)
เมื่อเข้าสู่หน้าทำข้อสอบ ระบบจะแสดงข้อมูลดังนี้:

* **Header:** มุมซ้ายบนแสดง **หมวดหมู่ (Category)** และ **ระดับความยาก (Difficulty)** ของข้อสอบชุดปัจจุบัน
* **Question:** คำถามจะปรากฏอยู่ตรงกลางอย่างชัดเจน (เช่น *What does "ephemeral" mean?*)
* **Answer Options:** มีตัวเลือกคำตอบ 4 ตัวเลือก ให้ผู้ใช้คลิกเลือกคำตอบที่คิดว่าถูกต้องที่สุดเพียง 1 ข้อ
* **Progress Bar:** ด้านล่างสุดจะแสดงลำดับข้อปัจจุบัน (เช่น *Question : 1/10*) เพื่อให้ทราบความคืบหน้า

---

### 3.4 การแสดงผลเฉลย (Answer Feedback)
ระบบจะทำการตรวจคำตอบทันทีที่ผู้ใช้เลือกตัวเลือก:

* **กรณีตอบถูก:** ปุ่มคำตอบที่เลือกจะเปลี่ยนเป็น **สีเขียว** พร้อมเครื่องหมายถูก (✓) แสดงว่าถูกต้อง
* **กรณีตอบผิด:** (ระบุตาม Logic โปรแกรม เช่น ปุ่มที่เลือกเป็นสีแดง และเฉลยข้อที่ถูกเป็นสีเขียว)
* ระบบจะพาไปยังข้อถัดไปโดยอัตโนมัติ หรือรอให้ผู้ใช้กดถัดไป (ขึ้นอยู่กับการตั้งค่า)

---

### 3.5 สรุปผลคะแนน (Quiz Completion)
เมื่อทำครบทุกข้อ ระบบจะแสดงหน้าสรุปผล:

* **Score:** แสดงคะแนนที่ทำได้ในรูปแบบตัวเลขและเปอร์เซ็นต์ (เช่น *8/10 (80%)*)
* **Message:** ข้อความให้กำลังใจตามระดับคะแนนที่ได้
* **ปุ่มควบคุม:**
    * **Play Again:** เพื่อเริ่มทำแบบทดสอบใหม่อีกครั้ง
    * **Back to Main Menu:** เพื่อกลับไปหน้าหลัก

---

## 4. การแก้ไขปัญหาเบื้องต้น (Troubleshooting)

* **เปิดโปรแกรมไม่ได้:** โปรดตรวจสอบว่าเครื่องคอมพิวเตอร์ได้ติดตั้ง Java เรียบร้อยแล้ว (สามารถตรวจสอบโดยพิมพ์ `java -version` ใน Command Line)
* **ไฟล์คำศัพท์ไม่โหลด:** ตรวจสอบว่าไฟล์ `vocab_data.json` วางอยู่ในโฟลเดอร์เดียวกับตัวโปรแกรม และมีโครงสร้างข้อมูลที่ถูกต้อง
