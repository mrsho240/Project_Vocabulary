# Scope Document: Vocabulary Quiz Generator

## 1. Project Overview (ภาพรวมโครงการ)
พัฒนาระบบสร้างแบบทดสอบคำศัพท์อัตโนมัติ (Vocabulary Quiz Generator) ที่ช่วยให้ผู้เรียนสามารถฝึกฝนคำศัพท์ภาษาอังกฤษผ่านการทำแบบทดสอบแบบปรนัย (Multiple Choice) โดยระบบจะทำการอ่านข้อมูลคำศัพท์จากไฟล์ภายนอกและสุ่มสร้างคำถามขึ้นมา

## 2. Objectives (วัตถุประสงค์)
1. เพื่อสร้างเครื่องมือช่วยจำคำศัพท์ที่มีประสิทธิภาพและใช้งานง่าย
2. เพื่อประยุกต์ใช้ความรู้เรื่องการจัดการไฟล์ (File I/O) และโครงสร้างข้อมูลในภาษา Java/Python
3. เพื่อเรียนรู้การจัดการ State ของแอปพลิเคชัน (การนับคะแนน, การเปลี่ยนข้อสอบ)

## 3. In-Scope (ขอบเขตที่ทำ)
ระบบจะครอบคลุมฟังก์ชันการทำงานดังนี้:
* **Data Import:** รองรับการอ่านไฟล์คำศัพท์จาก Text file (.txt) หรือ CSV (.csv) ที่มีรูปแบบ "คำศัพท์,คำแปล"
* **Quiz Generation:**
    * ระบบสุ่มคำถามจากรายการคำศัพท์ที่โหลดมา
    * สร้างตัวเลือก 4 ตัวเลือก (1 ถูก, 3 ผิด) โดยตัวเลือกหลอก (Distractors) จะสุ่มมาจากคำศัพท์อื่นในไฟล์เดียวกัน
* **Quiz Interface:** แสดงคำถามและตัวเลือกให้ผู้ใช้เลือกตอบผ่าน Command Line หรือ GUI (ตามที่กำหนด)
* **Scoring System:**
    * ตรวจคำตอบทันทีที่ผู้ใช้เลือก
    * เก็บสะสมคะแนน (Score) และจำนวนข้อที่ทำ
* **Feedback:** แสดงเฉลยเมื่อผู้ใช้ตอบผิด
* **Summary:** สรุปคะแนนรวมเมื่อจบการทดสอบ

## 4. Out-of-Scope (ขอบเขตที่ไม่ทำ)
ฟังก์ชันเหล่านี้จะไม่รวมอยู่ในเวอร์ชันแรก:
* ไม่มีระบบ Login หรือฐานข้อมูลผู้ใช้ (User Database)
* ไม่มีการบันทึกสถิติการทำข้อสอบย้อนหลังลงไฟล์ (History Log)
* ไม่รองรับการเพิ่ม/ลบ/แก้ไข คำศัพท์ภายในตัวโปรแกรม (ต้องแก้ที่ไฟล์ Source โดยตรง)
* ไม่มีการจับเวลาในการทำข้อสอบ (Timer)
* ไม่รองรับไฟล์รูปภาพหรือไฟล์เสียง
